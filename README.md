# Task 8: Hamburger Menu Layout

## Description
This task enhances the responsive landing page with a hamburger-style menu for smaller screens. The page focuses on a laundry services hero layout and mobile-friendly navigation.

## Features
- Desktop navigation with inline menu items and username link
- Hamburger menu button for smaller devices
- Accessible drop-down menu displayed when the button receives focus
- Responsive hero section with text and image content
- Styled action button for booking services

## Files
- `index.html` - markup for navigation, menu, and hero content
- `style.css` - responsive navigation, layout styling, and hamburger menu behavior

## Usage
1. Open `index.html` in a browser.
2. On desktop, view the full menu and hero section.
3. On narrow screens, use the `Menu` button to open a drop-down navigation list.

## Notes
- The hamburger menu is implemented in CSS using `:focus-within`.
- The page design is intended for both desktop and mobile layouts.
